# Hava Durumu Uygulaması – Python GUI

Python ve Tkinter kullanılarak geliştirilmiş, kullanıcı dostu bir hava durumu uygulamasıdır.  
Uygulama **OpenWeatherMap API** üzerinden gerçek zamanlı verileri alır ve şehir için sıcaklık, hava durumu açıklaması ve ikonlarını gösterir.

## Özellikler

- Kullanıcıların girdiği şehir için anlık hava durumu ve sıcaklık bilgisi  
- Hava ikonları ve görselleştirilmiş sıcaklık  
- “Hakkında” butonu ile geliştirici bilgisi  
- Windows için çalıştırılabilir `.exe` dosyası  
- Küresel şehir desteği (örn: Istanbul, London, New York, Tokyo)  

## Kullanılan Teknolojiler

- Python  
- Tkinter (GUI)  
- Requests (API isteği için)  
- Pillow (ikon görselleştirmesi için)  

## Python ile Çalıştırma

1. Python 3’ün yüklü olduğundan emin olun  
2. Gerekli kütüphaneleri yükleyin:

```bash
pip install requests pillow
```
3.Programı çalıştırın:
python main.py
