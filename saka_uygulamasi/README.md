# Rastgele Şaka Uygulaması – Python GUI

Python ve Tkinter kullanılarak geliştirilmiş basit ve kullanıcı dostu bir rastgele şaka uygulaması.  
Uygulama **JokeAPI** kullanarak gerçek zamanlı şakalar çeker ve Tkinter arayüzünde gösterir.

## Özellikler

- Rastgele şaka çekme (Tekli veya Setup-Delivery tipi)  
- Basit ve kullanıcı dostu Tkinter arayüzü  
- “Hakkında” butonu ile geliştirici bilgisi  
- Windows için çalıştırılabilir `.exe` dosyası  
- Tek butonla yeni şaka alma  

## Kullanılan Teknolojiler

- Python  
- Tkinter (GUI)  
- Requests (API isteği için)

## Python ile Çalıştırma

1. Python’un yüklü olduğundan emin olun (Python 3 önerilir)  
2. Gerekli kütüphaneyi yükleyin:
```bash
pip install requests
```
3.Programı çalıştırın:

python main.py
EXE İndirme
